 ---- SETP 1 ----
Make sure to install these dependensice -
	1. Python 2.7 (Make sure to add to PATH in Enviromental Variables)
	2. C++ Build Tools (http://go.microsoft.com/fwlink/?LinkId=691126)
	3. Node.js 

---- SETP 2 ----
Git clone the project to your computer either with 
git clone https://github.com/UBCMarsColony/SabatierDemoUI.git

OR

Use the GitHub Desktop app available to all OSs.

---- SETP 3 ----
cd into the SabatierDemoUI 
npm install
npm start
